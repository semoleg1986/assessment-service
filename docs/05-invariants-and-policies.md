# Инварианты И Политики

## Инварианты
1. В тесте должен быть хотя бы один вопрос.
2. Нельзя назначать тест архивированному/заблокированному ребенку.
3. У одного назначения не может быть больше одной активной попытки.
4. После `submit` ответы неизменяемы.
5. Расчет score должен быть детерминированным и воспроизводимым.

## Политики доступа
- Родитель работает только со своими детьми.
- Ребенок запускает/отправляет только свои назначения.
- Учитель/Админ создают тесты и читают агрегированные отчеты.

## Политика сигналов (MVP, rule-based)
- `norm`: >= 85% по кластеру микро-умений.
- `risk`: 70-84%.
- `gap`: 50-69%.
- `critical_gap`: < 50%.

Пороги конфигурируемые по предмету и классу.

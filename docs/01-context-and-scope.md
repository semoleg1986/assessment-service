# Контекст И Границы

## Контекст
Learning Assessment Context.

## Входит в контекст
- Каталог предметов/тем/микро-умений.
- Определение тестов (класс, предмет, набор вопросов).
- Жизненный цикл назначений тестов ребенку.
- Жизненный цикл попыток и отправка результатов.
- Сохранение ответов и расчет балла.
- Сигналы: норма, риск, пробел, критический пробел.

## Не входит в контекст
- Управление учетными записями и ролями.
- Владение профилем ребенка (User-Children Service).
- Уведомления.

## Зависимости
- User-Children Service: проверка существования/принадлежности ребенка.
- Auth Service: идентификация и роли акторов.
- Notification Service (опционально): события для уведомлений.

## Внешние интерфейсы
- HTTP API (`/v1/...`) для BFF и admin-инструментов.
- События (опционально) для аналитики и уведомлений.
